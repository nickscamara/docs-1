The `connector_id` for Zendesk is `zendesk`.

Psychic supports ingesting tickets from Zendesk's [Ticketing APIs](https://developer.zendesk.com/api-reference/ticketing/introduction/) and articles from [Help Center APIs](https://developer.zendesk.com/api-reference/help_center/help-center-api/introduction/)

## Authorization Type
Psychic uses [OAuth 2.0](https://developer.zendesk.com/documentation/ticketing/working-with-oauth/creating-and-using-oauth-tokens-with-the-api/) to connect to Zendesk.

## Configuration

### Custom OAuth Credentials
The Zendesk connector requires the end user to know the subdomain of the Zendesk account they want to connect to.

## Synchronization
Zendesk Documents can be fetched as-neded from the `/get-documents` endpoint. 
It's also possible to set up 24 hour syncs by providing a webhook URL in the [dashboard](https://dashboard.psychic.dev/syncs)
