You can access the dashboard at [dashboard.psychic.dev](https://dashboard.psychic.dev).

Things you can do:
* View and manage all the connections across all connectors
* Connect to your own data through the Psychic Link [playground](https://dashboard.psychic.dev/playground)
* Customize the [Psychic Link](/psychic-link) modal with your own logo and name
* [Get API keys](https://dashboard.psychic.dev/api-keys) to use for creating new connections and querying data
* Configure connectors to use your own OAuth credentials
